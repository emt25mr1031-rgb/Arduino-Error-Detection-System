# Arduino Error Detection System

This project detects sensor faults using Arduino and provides alert using LEDs, LCD, and buzzer.

## Components Used
- Arduino UNO
- Sensor
- LCD Display
- LEDs
- Buzzer

## Features
- Real-time fault detection
- Visual LED indicators
- Audible buzzer alert
- LCD status display

## Simulation
Created using Wokwi Simulator.
